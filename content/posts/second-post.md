## 设计与功能

好的设计应该是隐形的。它不会喧宾夺主，而是让用户自然地完成任务。

### 暗色模式

暗色模式不只是把背景变黑。它需要精心设计层次和对比度。

### 工业美学

《明日方舟：终末地》的工业美学给人留下深刻印象。它的设计语言有几个特点：

- **硬朗的几何线条** - 干净利落，不拖泥带水
- **高对比度强调色** - 在暗色背景上特别醒目
- **质感的处理** - 纹理和细节让界面更有层次

```css
/* 模拟 Endfield 风格的斜线纹理 */
.element {
  background-image: repeating-linear-gradient(
    -45deg,
    transparent,
    transparent 2px,
    rgba(255, 255, 255, 0.03) 2px,
    rgba(255, 255, 255, 0.03) 4px
  );
}
```

## 小结

设计是一门平衡的艺术。找到功能与美学的平衡点，需要持续地学习和实践。
